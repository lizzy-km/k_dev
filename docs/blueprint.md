# **App Name**: Drop It

## Core Features:

- Voice Recording: Record audio clips piece by piece.
- Audio Upload: Upload audio files (wav, mp3, etc.) from local device.
- Audio Arrangement: Arrange audio clips rhythmically.
- Clip Animation: Associate unique animations and characters with each audio clip.
- User Accounts: Enable multiple users and accounts, saved to the device.
- User Creations: Browse audio creations from other users stored locally.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a creative and modern feel.
- Background color: Light lavender (#F3E5F5) to complement the purple.
- Accent color: Bright pink (#E91E63) for highlights and interactive elements.
- Body and headline font: 'Poppins' (sans-serif) for a clean, geometric style suitable for headings and body text
- Use vibrant, custom icons that reflect audio and animation themes.
- Modern and intuitive layout for arranging audio clips.
- Smooth, engaging animations to enhance user experience during recording and playback.
