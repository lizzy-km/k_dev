# **App Name**: Kaung's Portfolio

## Core Features:

- Skills Showcase: Display technical and soft skills with proficiency levels.
- Project Portfolio: Showcase projects with descriptions, links to live demos, and source code (GitHub).
- Experience Timeline: Visualize work experience and education in a clear, chronological timeline.
- Contact Form: Enable visitors to send messages directly.
- Figma to Code Conversion tool: Integrate Figma to React functional component code conversion using an LLM tool. Users can copy-paste from the figma plugin.

## Style Guidelines:

- Primary color: Dark grayish-blue (#393E46) for a professional feel, aligning with provided color preferences.
- Background color: Very dark grayish-blue (#222831), almost black, to provide strong contrast and a modern look.
- Accent color: Grayish-yellow (#DFD0B8) for highlights and interactive elements, drawn from the provided palette, creating contrast and visual interest.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern aesthetic.  Note: currently only Google Fonts are supported.
- Use simple, line-based icons for skills, project categories, and navigation.
- Employ a grid-based layout for content organization, ensuring responsiveness across devices.
- Use subtle transition animations for section scrolling and project details.
